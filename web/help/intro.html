SIREN — Interactive and visual geospatial redescription mining

Introduction


Redescription mining is a powerful data analysis tool that aims at finding alternative descriptions the same entities.
For example, in biology, an important task is to identify the bioclimatic constraints that allow some species to survive, that is, to describe geographical regions in terms of both their bioclimatic conditions and the fauna that inhabits them.
When the entities are geographic locations, we qualify the redescriptions as geospatial.

Siren is a tool for interactive mining and visualization of geospatial redescriptions. It is based on the ReReMi mining algorithm.
